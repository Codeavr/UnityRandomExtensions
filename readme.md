## Install

Find `Packages/manifest.json` in your project and edit it to look like this:
```json
{
  "dependencies": {
    "com.codeavr.unity-random-extensions": "https://github.com/Codeavr/UnityRandomExtensions.git"
  }
}
```


### Requirements

* Unity any version
* Git
